# cryptoapp
Cryptoapp is a Bitcoin price alert app that will allow a user to set a price in order to receive a native desktop notification if the current price of Bitcoin exceeds the specified amount.


## Technologies:
- ElectronJS
- Node
- coinmarketcap api
